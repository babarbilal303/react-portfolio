### Description

A simple portfolio template for developer/designers built with React. 

### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap
- Edit Content From One Place

### Setup

Get the code

<pre>git clone git@bitbucket.org:babarbilal321/react-portfolio.git</pre>
 
Install required dependencies

<pre>yarn install</pre>


Start the server

<pre>yarn start</pre>
